# 插件文档目录

本目录用于存放插件的详细文档，包括：

- 插件使用说明
- 安装配置指南
- 常见问题解答
- 开发文档等

## 文档结构

```
docs/
├── plugin_name/
│   ├── installation.md      # 安装说明
│   ├── configuration.md     # 配置说明
│   ├── usage.md            # 使用说明
│   ├── troubleshooting.md  # 故障排除
│   └── changelog.md        # 更新日志
└── README.md
```

## 文档规范

1. 使用 Markdown 格式
2. 包含详细的配置说明
3. 提供使用示例
4. 标注注意事项
5. 维护更新日志
